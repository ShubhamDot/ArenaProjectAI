# Contributing to Arena AI

Thank you for considering contributing! Here's how you can help.

## Reporting Bugs

1. Open an issue on GitHub
2. Clearly describe the issue including steps to reproduce
3. Mention your Android version and device model
4. Include screenshots if applicable

## Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test on a physical device or emulator
5. Submit a pull request with a clear description

## Code Style

- Java with standard Android conventions
- No third-party libraries without discussion
- Keep the APK size under 3 MB
